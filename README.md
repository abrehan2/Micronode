# Micronode  
**Docker • RabbitMQ • MongoDB**

A simple, production-style microservices architecture using **Node.js**. Each service is containerized with **Docker**, communicates asynchronously via **RabbitMQ**, and stores data in **MongoDB**.

---

## ✅ Features

- Multiple independent Node.js microservices
- Inter-service communication through RabbitMQ (AMQP)
- MongoDB databases per service
- Docker Compose for one-command startup
